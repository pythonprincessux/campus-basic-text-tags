Basic HTML Text Tags

Beginning practice with the basics

Comments are used to add additional information to your code that will not be displayed on the page. They are helpful for other developers and for reminding yourself what different parts of your code do.

Basic HTML text tags help structure your content and make it easier to read and understand.

Marking up Text in Blocks

When you have text that is grouped together, you should use block-level elements such as paragraphs and sections to organize it.

Marking Up Text to Convey Meaning

Sometimes, you are marking up text to convey meaning rather than structure. For example, you might use strong importance to emphasize critical content or emphasis to subtly stress a phrase.

Inline code such as <p>, <section>, <header>, <footer>, and <main> is useful when discussing HTML elements directly.

Sometimes text is bolded for visual attention or italicized for stylistic reasons. You may also want to highlight key ideas, important notes, or special terms for readers.
